## tottori-municipalities.svg

- Source: MLIT National Land Numerical Information N03 Administrative Area Data, Tottori Prefecture, 2024-01-01.
- URL: https://nlftp.mlit.go.jp/ksj/gml/data/N03/N03-2024/N03-20240101_31_GML.zip
- License: Public Data License 1.0 (PDL1.0), unless otherwise stated by the National Land Numerical Information site.
- Acquisition date: 2026-06-11
- Generated by: `scripts/generate_tottori_municipality_map.py`
- Source format used: the GeoJSON bundled in the N03 GML zip.
- Simplification: Ramer-Douglas-Peucker tolerance `0.0015` degrees, minimum projected ring area `0.04`.
- Grouping: polygons are grouped by `N03_007`; each municipality is emitted as one SVG `<g>` click region.
- Use in this site: Tottori prefecture page navigation. Four cities are active; other municipalities are shown as unsupported.
